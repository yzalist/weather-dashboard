# Weather Dashboard - 设计理念

## 选定设计方向：现代极简主义 + 气象数据美学

### 设计哲学
**Modern Minimalism with Data Visualization Focus**

这是一个融合**极简设计**与**数据美学**的天气应用。设计目标是让复杂的气象数据通过优雅的视觉语言变得易于理解，同时保持界面的清爽与专业感。

---

## 核心设计原则

### 1. **极简但不空洞**
- 移除所有不必要的装饰，保留功能本质
- 使用充足的负空间（whitespace）来呼吸
- 每个元素都有明确的信息传递目的

### 2. **数据驱动的视觉层级**
- 温度、湿度、风速等关键数据通过大号字体和颜色突出
- 使用色彩心理学：红色表示热，蓝色表示冷
- 图表和图形化展示复杂数据

### 3. **自然与科技的融合**
- 天气本身具有自然属性，设计应融入自然元素
- 使用柔和的渐变背景模拟天空
- 微妙的动画模拟风、云、雨等自然现象

### 4. **可访问性优先**
- 高对比度确保易读性
- 清晰的信息结构，快速扫描
- 响应式设计支持所有设备

## 色彩哲学

### 主色板
| 用途 | 颜色 | OKLCH | 含义 |
|------|------|-------|------|
| 晴天背景 | 浅蓝 | `oklch(0.85 0.15 250)` | 清爽、开放、希望 |
| 阴天背景 | 灰蓝 | `oklch(0.75 0.08 250)` | 沉静、稳定 |
| 雨天背景 | 深蓝灰 | `oklch(0.65 0.1 250)` | 沉重、压抑 |
| 高温强调 | 暖橙红 | `oklch(0.65 0.25 30)` | 热、警告 |
| 低温强调 | 冷蓝 | `oklch(0.55 0.2 250)` | 冷、清爽 |
| 文字主色 | 深灰 | `oklch(0.25 0.01 0)` | 可读性、专业 |

### 背景渐变
- **晴天**：从浅蓝顶部到白色底部（模拟天空）
- **多云**：灰蓝色均匀背景
- **雨天**：深蓝灰渐变，略显压抑

## 布局范式

### 非对称网格结构
- **左侧**：当前城市的大型天气卡片（占 60%）
- **右侧**：城市列表 + 快速信息面板（占 40%）
- **底部**：7天预报水平滚动条

### 信息架构
```
┌─────────────────────────────────┐
│  Header (搜索 + 设置)             │
├──────────────┬──────────────────┤
│              │                  │
│  主天气卡    │  城市列表 +      │
│  (大字体)    │  快速面板        │
│              │                  │
├──────────────┴──────────────────┤
│  7天预报 (水平滚动)             │
├──────────────────────────────────┤
│  详细数据 (湿度、风速、气压等)   │
└──────────────────────────────────┘
```

## 标志性元素

### 1. **动态天气图标**
- 使用 SVG 动画，而非静态图标
- 晴天：太阳缓慢旋转
- 下雨：雨滴下落动画
- 多云：云朵漂移

### 2. **渐变卡片**
- 主天气卡片采用微妙的渐变背景
- 根据天气类型动态改变渐变方向和颜色

### 3. **数据可视化图表**
- 使用 Recharts 绘制温度曲线、湿度柱状图
- 柔和的色彩，避免过度饱和

---

## 交互哲学

### 微交互
- **悬停效果**：卡片轻微上升，阴影加深
- **过渡动画**：所有状态变化都有 300-500ms 的平滑过渡
- **加载状态**：骨架屏或脉冲加载动画

### 导航流程
1. 首屏显示默认城市（或用户上次查看的城市）
2. 点击城市列表快速切换
3. 搜索框支持模糊搜索
4. 侧滑或点击查看详细信息

### 响应式设计
- **移动端**：单列布局，卡片全宽
- **平板**：两列布局
- **桌面**：三列布局（左主卡片 + 右侧栏 + 详情面板）

---

## 动画指南

### 页面加载
- 背景渐变 300ms 淡入
- 主卡片从下方 200ms 滑入
- 城市列表依次 100ms 错开出现

### 天气更新
- 温度数字变化时，数字颜色闪烁 1 次
- 图标平滑过渡到新天气状态

### 用户交互
- 按钮点击：缩放 0.95 → 1.0（100ms）
- 卡片悬停：translateY(-4px)，阴影加深
- 搜索框获焦：边框颜色变为主色，阴影出现

## 排版系统

### 字体选择
- **标题字体**：`Geist` 或 `Inter` (粗体 700)
- **正文字体**：`Inter` (常规 400)
- **数据字体**：`Courier New` (等宽，用于温度等精确数据)

### 字号层级
| 用途 | 大小 | 权重 | 行高 |
|------|------|------|------|
| 页面标题 | 48px | 700 | 1.2 |
| 城市名 | 32px | 600 | 1.3 |
| 温度 | 72px | 700 | 1 |
| 天气描述 | 18px | 400 | 1.5 |
| 数据标签 | 14px | 500 | 1.4 |
| 正文 | 16px | 400 | 1.6 |

## 实现细节

### CSS 变量（index.css）
```css
--primary: oklch(0.55 0.2 250);      /* 冷蓝 */
--accent: oklch(0.65 0.25 30);       /* 暖橙红 */
--background: oklch(0.98 0.001 0);   /* 几乎白色 */
--foreground: oklch(0.25 0.01 0);    /* 深灰 */
--radius: 16px;                      /* 圆角 */
```

### 组件样式
- **卡片**：`rounded-2xl shadow-lg hover:shadow-xl transition-shadow`
- **按钮**：`rounded-lg px-4 py-2 transition-all hover:scale-105`
- **输入框**：`rounded-lg border-2 border-border focus:border-primary`
